# XMPP project
# Description
- Chat using XMPP protocol
- Python 3.10.5
- Educational propuses
- Understand the asynchronous programming foundations required 
# Functionalities
- Notifications of messages
- Private message
- Group message
- View contacts/Contact
- Login/Logout
- Creation of account
# How to run this?
```
pip install slixxmpp==1.8.1
pip install xmpppy 
python client.py
```
